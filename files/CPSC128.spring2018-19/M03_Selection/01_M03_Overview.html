Week 3 Overview

So far we've learned to write SIPO (Sequential Input, Processing, and Output) programs, but it is rare that we know ahead of time exactly what processing will be required to solve a problem. Instead it usually depends on the situation and that means we have to select what code to execute depending on the values that are input.

The core skill to develop as you work through this module is to be able to use the if statement to implement selection in your programs.

A secondary topic will be controlling the appearance of your output using format codes, but this is just a matter of details.

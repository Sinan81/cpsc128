Identifying the necessary classes

We can learn a lot about what code we have to write by carefully examining our specification code. First let's look for calls to constructors. The form of a constructor is that it uses a class name as a function. We've seen examples of this with Python's built-in classes, e.g.

>>> num = int(43.72)
>>> num
43
>>> lst = list('Tim')
>>> lst
['T', 'i', 'm']
>>>
